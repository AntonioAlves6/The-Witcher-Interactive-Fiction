# The-Witcher-Interactive-Fiction

This repository contains a simple game using the IF language. This was a game developed for the Multimedia Content Production course in Instituto Superior Técnico.

The objective was to have a witty writing, an iteresting atmosphere, and creativity. Include at least four rooms, at least on puzzle, a rich enough vocabulary (think synonyms, several verbs a player could conceivably try, etc.). Is interesting and fun! 

So this game is an adapation of the first tale of the first book in The Witcher series written by Polish author Andrzej Sapkowski. To adapt the tale, I decided to shorten it a little while staying almost true to the original. Don't expect it to be exactly the same... I also wanted to give the player a sense of exploring and making decisions that change future actions. 

In total, there are 10 rooms. Some are just for passing through and examining stuff, which Geralt does all the time. The puzzle is a boss fight, following the original material. I designed it so that if the players explore everything and talk with everyone, hints are printed and it should be pretty straightforward. I also included a secret "walkthrough" where a player can finish the game without exploring that much, but for this, they need to know the lore and the story beforehand, or be lucky! I also included small details: for example, right at the start of the game, the player can walk into the first inn, but is always kicked out. If the player tries to enter more than 3 times, they are killed. Or, when checking items of a certain character before talking with him, he says something. If you insist, the character can become quite aggressive, which also adds a line to the dialogue the player is supposed to have. In the fight you have a lot of options, like drinking potions before or during the fight, which can change the narrative. Well, if you like it, I encourage you to try different things to check it out, and then if you really like it, read the books! 

I encourage you to read carefully everything the characters say, since the directions and small details, for example, on how to break the curse, are told by them. During the puzzle, there are hints, which I added later since I thought people could get frustrated. And I understand that it is not a traditional puzzle... but I think it delivers something similar!

I also posted the game on a site. You can play it [here](https://dbsgtcp4.play.borogove.io/).

Finally there is the compiled file, the source code, and the walkthrough with two options: the intended solution and the secret solution, if you want to try it! Hope you like it!
